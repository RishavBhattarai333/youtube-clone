# VideoGram 

A YouTube clone built with React, TypeScript, and Vite.

## Live Demo
[videogram-rishav.vercel.app](https://videogram-rishav.vercel.app)

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Firebase Auth
- YouTube Data API v3

## Features
- Search videos
- Watch videos
- Firebase authentication
- Channel pages
- Responsive design
